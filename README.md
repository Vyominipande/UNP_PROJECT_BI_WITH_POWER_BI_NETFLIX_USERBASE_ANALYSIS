# UNP_PROJECT_BI_WITH_POWER_BI_NETFLIX_USERBASE_ANALYSIS

I did this project in collaboration with UNP, where the primary objective was to analyze Netflix userbase data and generate meaningful business insights using Power BI. This project focuses on understanding customer behavior, subscription preferences, revenue performance, and churn risk in a streaming service environment.
The dashboard includes multiple report pages such as KPI Summary, Revenue Analysis, User Base Analysis, Churn Risk Analysis, and Business Recommendations.
Key metrics analyzed in this project include total users, total revenue, ARPU, customer lifetime value, and churn risk percentage.
I created interactive visualizations such as bar charts, line charts, donut charts, KPI cards, and trend analysis reports for better decision-making. 
Dynamic slicers were implemented for country, device type, gender, tenure category, and subscription plan to allow easy data exploration. Overall, this project demonstrates how Power BI can transform raw customer data into actionable insights to improve retention strategies, optimize subscription plans, and drive revenue growth.
